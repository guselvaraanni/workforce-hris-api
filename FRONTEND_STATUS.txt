🎨 THEME SYSTEM
─────────────────────────────────────────────────────────────────────────────
✅ Dark Mode (Default)
   - Professional dark interface
   - Easy on the eyes
   - Click sun ☀️ to switch

✅ Light Mode
   - Clean white interface
   - High contrast
   - Click moon 🌙 to switch

✅ Persistent Storage
   - Theme preference saved in localStorage
   - Remembers your choice

🔌 API INTEGRATION
─────────────────────────────────────────────────────────────────────────────
✅ Authentication
   - JWT Token-based auth
   - Bearer token headers
   - Automatic refresh on 401

✅ Data Fetching
   - GET /employees/ → Employee list
   - GET /departments/ → Departments
   - GET /leave-requests/ → Leave requests
   - GET /bulk-uploads/ → Upload history
   - GET /audit-logs/ → Audit trail

✅ Data Submission
   - POST /employees/ → Create employee
   - DELETE /employees/{id}/ → Remove employee
   - POST /bulk-uploads/ → Upload CSV

❓ TROUBLESHOOTING
─────────────────────────────────────────────────────────────────────────────
Issue: Pages show "Loading..." forever
Fix:   Make sure backend is running (python manage.py runserver)

Issue: "Connection error" on login
Fix:   Check API is accessible at http://localhost:8000/api/v1/health/

Issue: Theme not saving
Fix:   Clear localStorage (F12 → Application → Local Storage → Clear)

Issue: Unauthorized errors
Fix:   Login again, token may have expired
